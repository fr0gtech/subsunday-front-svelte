# subsunday-front

The frontend for [sub-sunday.com](https://sub-sunday.com)

This displays sub sunday voting data for [lirik](https://twitch.tv/lirik)

## Features

- Display games by period sorted by votes
- Game page
- User page
- Realtime vote updates with sockets

## Developing

Clone this repo and install dependencies:

```bash
pnpm install
```

Run dev build

```bash
pnpm dev
```
